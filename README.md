# Footy: Machine Learning Model

## Stack

Backend
- Python3
- Packages: Pandas, Numpy, Sci-kit Learn

## Usage

Clone locally and install packages.

Backend: 
1. ```python3 -m venv footy-env```
2. ```source footy-env/bin/activate```
3. ```pip install -r requirements.txt```
4. ```python main.py```
